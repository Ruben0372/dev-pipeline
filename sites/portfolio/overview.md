# Portfolio

## Summary

Security-focused full stack developer portfolio and freelance platform hub. Personal brand site for Upwork/Fiverr/LinkedIn that showcases Ruben's security engineering and full-stack work. Built with Next.js 15, Tailwind CSS v4, Framer Motion, and shadcn/ui + 21st.dev components.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript 5.7
- **Styling:** Tailwind CSS v4, shadcn/ui, 21st.dev components
- **Animation:** Framer Motion 12
- **Icons:** Lucide React
- **Deploy:** Vercel
