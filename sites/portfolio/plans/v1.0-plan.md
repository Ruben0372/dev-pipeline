## Key Decisions Made During Build

| Decision | Choice | Rationale |
|----------|--------|-----------|
| Smooth scroll | Native `window.scrollTo` | Lenis caused conflicts with programmatic scrolling |
| Nav indicator | CSS opacity transition | Framer Motion `layoutId` FLIP reset scroll position |
| Hero animation | Parent `variants` propagation | Individual `initial`/`animate` replayed on re-render |
| Scroll spy guard | `isScrolling` module flag | IntersectionObserver re-renders fought scroll position |
| Content layer | next-mdx-remote + gray-matter | Simple, no build-time dependency like contentlayer |
